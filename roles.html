Gestion des Rôles Utilisateurs

4. Gestion des rôles des utilisateurs

Le système de gestion des rôles permet de définir précisément les permissions de chaque type d'utilisateur, garantissant ainsi la sécurité des données et le bon fonctionnement des processus métiers.

Administrateur

Le rôle administrateur dispose de tous les droits sur le système. Il peut :

  • Créer, modifier et supprimer des utilisateurs
  • Configurer tous les paramètres système
  • Accéder à toutes les fonctionnalités sans restriction
  • Générer des rapports complets
  • Modifier les permissions des autres rôles

Recommandations :

Ce rôle doit être attribué uniquement aux personnes responsables de l'administration globale du système.

Responsable des stocks

Le responsable des stocks peut gérer tous les aspects liés aux produits et aux dépôts :

  • Alimenter les dépôts en produits
  • Effectuer des transferts entre dépôts
  • Consulter les niveaux de stock en temps réel
  • Générer des rapports de stock
  • Gérer les bons de ravitaillement

Recommandations :

Ce rôle est idéal pour les responsables logistiques et les gestionnaires de dépôts.

Opérateur

L'opérateur est le rôle standard pour les utilisateurs quotidiens du système :

  • Saisir les consommations des moteurs
  • Générer des rapports basiques
  • Consulter les historiques de consommation
  • Visualiser les niveaux de stock
  • Gérer les cartes Naftal qui lui sont attribuées

Recommandations :

Ce rôle convient aux techniciens et opérateurs terrain qui doivent enregistrer les consommations.

Consultant

Le consultant a un accès en lecture seule à certaines fonctionnalités :

  • Consulter les rapports générés
  • Visualiser les tendances de consommation
  • Accéder aux historiques (sans modification)
  • Exporter des données pour analyse

Recommandations :

Ce rôle est adapté aux auditeurs, consultants externes et managers qui ont besoin d'accéder aux données sans les modifier.

Gestion des utilisateurs

La création et la gestion des utilisateurs se fait exclusivement par les administrateurs via une interface dédiée.

Création d'un utilisateur

  1. Accéder au menu "Gestion des utilisateurs"
  2. Cliquer sur "Nouvel utilisateur"
  3. Remplir les informations requises
  4. Attribuer un rôle
  5. Enregistrer

Modification d'un utilisateur

  1. Sélectionner l'utilisateur dans la liste
  2. Cliquer sur "Modifier"
  3. Apporter les modifications nécessaires
  4. Enregistrer les changements

Désactivation d'un utilisateur

  1. Sélectionner l'utilisateur dans la liste
  2. Cliquer sur "Désactiver"
  3. Confirmer la désactivation

Note: Les utilisateurs ne sont pas supprimés mais désactivés pour conserver l'historique.
