Guide d'Installation

7. Installation (Mode Solo)

Pour installer l'application en mode Solo, veuillez suivre les étapes ci-dessous :

1

7.1 Téléchargement des fichiers requis

  • Base de données locale : Télécharger ici
  • Framework .NET : Télécharger ici
  • Fichier d'installation (Setup) : Télécharger DB ici , Setup ici

NB: En cas de multi mode nécessitant une configuration personnalisée, contacter la elhoussam

2

Installation de la base de données

  1. Exécutez le fichier d'installation de la base de données
  2. Suivez les instructions de l'assistant d'installation
  3. Notez le chemin d'installation pour la configuration ultérieure
3

Installation du Framework .NET

  1. Exécutez le programme d'installation de .NET Framework
  2. Acceptez les termes de licence
  3. Attendez la fin de l'installation (peut prendre plusieurs minutes)
4

Installation de l'application

  1. Exécutez le fichier setup.exe
  2. Choisissez le répertoire d'installation
  3. Configurez le chemin d'accès à la base de données
  4. Complétez l'installation

Dépannage

Erreur de connexion à la base de données

Vérifiez que le service de base de données est démarré et que le chemin d'accès est correct.

Framework .NET manquant

Assurez-vous d'avoir installé la version requise de .NET Framework avant d'installer l'application.

Problèmes de permissions

Exécutez l'installation en tant qu'administrateur si vous rencontrez des erreurs d'accès.
